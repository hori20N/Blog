##### **1 删除链表中的节点**
```java
  public ListNode removeElements(ListNode head, int val) {

          ListNode HEAD = new ListNode(-1);
              HEAD.next = head;
          ListNode prev = HEAD;
          ListNode curr = head;

          while(curr != null){
              if(curr.val == val){
                  prev.next = curr.next;
              }else{
                  prev = curr;
              }
              curr = curr.next;
          }
          return HEAD.next;
    }
```
