##### **1. 删除链表中的节点（困难）**
```java
  public ListNode removeElements(ListNode head, int val) {

          ListNode HEAD = new ListNode(-1);
              HEAD.next = head;
          ListNode prev = HEAD;
          ListNode curr = head;

          while(curr != null){
              if(curr.val == val){
                  prev.next = curr.next;
              }else{
                  prev = curr;
              }
              curr = curr.next;
          }
          return HEAD.next;
    }
```

```java
  public ListNode deleteDuplicates(ListNode head) {

          // bugfix #1: 空表处理
          if(head == null){
              return head;
          }

          ListNode headCurr = head.next;

          while(headCurr != null){

                  headPrev.next = headCurr;

          }

      }
```

##### **3. 合并两个有序链表（困难）**
1）方法一（将两个链表插入到一个新链表)
```java
  public ListNode mergeTwoLists(ListNode l1, ListNode l2) {

          ListNode RESULT = new ListNode(-1);
          ListNode currentNode = RESULT;

          while(l1 != null && l2 != null){
              if(l1.val > l2.val){
                  currentNode.next = l2;
                  l2 = l2.next;
              }else{
                  currentNode.next = l1;
                  l1 = l1.next;
              }
              currentNode = currentNode.next;
          }

          if(l1 != null){
              currentNode.next = l1;
          }
          if(l2 != null){
              currentNode.next = l2;
          }
          return RESULT.next;
    }
```
2) 方法二（真正合并两个链表，不建立新的链表。PS: 该方法实现太复杂了，空处理、两个Flag、六个辅助变量。有必要思考一下为啥这么麻烦）
```java
  public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
        if(l1 == null && l2 != null){
            return l2;
        }else if(l1 != null && l2 == null){
            return l1;
        }else if (l1 == null && l2 == null){
            return null;
        }

        ListNode FLAG_Prev = new ListNode(-2);
        ListNode FLAG = new ListNode(-1);
        FLAG.next = l1;
        FLAG_Prev.next = FLAG;
        ListNode prev = FLAG;
        ListNode curr = l1;
        ListNode curr0 = l1;
        ListNode insert = l2;
        while(curr != null && insert != null){
            if(insert.val < curr.val){
                ListNode end0 = insert;
                ListNode end1 = insert.next;
                while(end1 != null && end1.val < curr.val){
                    end0 = end1;
                    end1 = end1.next;
                }
                prev.next = insert;
                insert = end0.next;
                end0.next = curr;
            }
            prev = curr;
            curr0 = curr;
            curr = curr.next;
        }

        if (insert != null){
            curr0.next = insert;
        }
        return FLAG_Prev.next.next;
    }
```

##### **4. 反转链表（困难）**
1）raw 版
```java
  public ListNode reverseList(ListNode head) {

      if(head == null){
          return null;
      }
      ListNode nextNode = head;
      ListNode Flag = new ListNode(-1);
      int i = 0;
      while(head.next != null){
          if(i == 0){
              nextNode = head.next;
              Flag.next = head;
              head.next = null;
          }else{
              nextNode = head.next;
              head.next = Flag.next;
              Flag.next = head;
          }
          head = nextNode;
          i++;
      }
      nextNode.next = Flag.next;
      Flag.next = nextNode;
      return Flag.next;
  }
```
2）light版
```java
  public ListNode reverseList(ListNode head) {
      if(head == null){
          return null;
      }
      ListNode pre = null;
      ListNode nex = null;
      while(head != null){
          nex = head.next;
          head.next = pre;
          pre = head;
          head = nex;
      }
      return pre;
  }
```
3) 递归版
```java
  public ListNode reverseList(ListNode head) {
      if(head == null || head.next == null){
          return head;
      }
      return reverse(null, head);
  }
  public ListNode reverse(ListNode pre, ListNode suc){
      if(suc == null)  return pre;
      ListNode res = reverse(suc, suc.next);
      suc.next = pre;
      return res;
  }
```
##### **5. 输出链表的中间结点 （简单）**
```java
public ListNode middleNode(ListNode head) {
    ListNode probeNode = head;
    ListNode resultNode = head;
    while(probeNode != null){
        if(probeNode.next == null){
            return resultNode;
        }
        if(probeNode.next.next == null){
            return resultNode.next;
        }
        resultNode = resultNode.next;
        probeNode = probeNode.next.next;
    }
    return resultNode;
}
```

```java
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        int lenA = listLength(headA);
        int lenB = listLength(headB);
        ListNode newHead = null;
        int padding = lenA-lenB;
        if(padding > 0){
            newHead = fixHead(headB, padding);
            return getCommonNode(headA,newHead);
        }else if(padding < 0){
            newHead = fixHead(headA, padding);
            return getCommonNode(newHead,headB);
        }else{
            return getCommonNode(headA,headB);
        }
    }
    public ListNode getCommonNode(ListNode headA, ListNode headB){

        while(headA != null){
            if(headA == headB){
                return headA;
            }
            headA = headA.next;
            headB = headB.next;
        }
        return null;
    }

    public int listLength(ListNode head){
        int len = 0;
        while(head != null){
            len++;
            head = head.next;
        }
        return len;
    }
    public ListNode fixHead(ListNode head, int len){
        if(len == 0){
            return head;
        }
        ListNode Flag = new ListNode(-1);
        ListNode tempTail = new ListNode(-1);
        ListNode tempHead = tempTail;
        Flag.next = tempHead;
        while(len > 1){
            ListNode newFake = new ListNode(-1);
            newFake.next = tempHead;
            tempHead = newFake;
            Flag.next = tempHead;
            len--;
        }
        tempTail.next = head;
        return Flag.next;
    }
```
